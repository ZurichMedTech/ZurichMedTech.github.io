## Landing Page

<p align="center">
  <img width="90%" src="assets/landingpage.png">
</p>

The user can log in **S4L lite** platform via this landing page. Two-factor authentication (2FA) is required:
  - a user email address and password,
  - an SMS with the required one-time 4-digit pin is sent to the user's mobile phone number,
  - for users who are having difficulties of receiving SMS messages, the code can be requested via email.

Registration is currently available only via an invitation and clicking ```Create Account``` button will show the user the webpage link (https://zmt.swiss/support/support/s4l-light-download/) to request access to **S4L lite**.

If the user cannot remember the password of an existing account, it can be reset by clicking the ```Forgot Password?``` and following the instructions there.
